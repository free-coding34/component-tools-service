# 技术选型
- 服务器：nodejs
- 数据库：mysql

# 项目结构
```
├── README.md                       // help
├── config                          // 业务相关配置文件
    ├── db.js                       // 数据库配置
├── db                              // 数据库查询
    ├── index.js                    
├── router                          // 不同模块的路由处理
    ├── index.js                    // 基础服务
├── sevice                          // 业务处理 
├── utils                           // 工具类
├── package.json
```

# 实现的功能
1. 初始化
   - 目录结构定义
   - web 服务器搭建
   - 数据库配置
   - 环境配置
2. 获取表列表
3. 获取字段名列表
4. 获取模板列表
5. 模板存储
6. 生成模板文件
